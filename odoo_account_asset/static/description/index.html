Account Asset Management for Odoo Community Edition

This app allow user to manage accouning assets and depreciations. You will be allow to configure asset types/categories. Depreciation will be computed as per settings and journal entry for depreciation will be created.

Main Features:

  • Allow you to set asset type on product form.
  • System will create asset automatically if vendor bill will be set with Asset type/category. So on validation of vendor bill system will create asset automatically.
  • Account Asset Types menu for setting up your categories of assets.
  • Assets list and form views.
  • Assets Report.
  • View Assets Report with Depreciation Date & Vendor.
  • View Bar Chart, Line Chart & Pie Chart for Assets.
  • For more details please watch video and screenshots.
  • Only for Odoo Community Edition.
