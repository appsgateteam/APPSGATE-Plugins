Car Service Booking Management

This module add feature to book car service and process on them!

Key Features:

  • Define Service Packages by Car or Globally
  • Book Service by Slots
  • Generate tasks based on service
  • Print service order in pdf
  • Add extra consumption charge during service
  • Book car service from website page
